# CHANGELOG — 鐵砧桌面放置遊戲

## v0.5.5

### 🌐 多人模式 — 新功能
- 自動重連：被伺服器強制斷線後，每 60 秒自動重試，最多 10 次；失敗後通知「請聯繫一加」
- 進階設定：IP、Port 輸入框移至進階設定，主畫面更簡潔；Port 預設改為 9527
- 動畫補幀（lerp）：補幀開關，讓對方鎚子動作更流暢，預設開啟
- 伺服器公告：管理員可向所有玩家發送公告，以托盤氣泡顯示
- 伺服器 GUI：合併選單、開啟/關閉按鈕、Port 設定移入設定對話框

### 🪟 多人模式 — 體驗改善
- 記住對方鐵砧狀態：位置、縮放、隱藏等設定自動儲存，重新加入房間後原位還原
- 托盤「所有鐵砧移回中央」：一鍵把所有人的鐵砧移回螢幕中央
- 對方鐵砧預設縮放 75%，名字預設 hover 才顯示（可右鍵「固定顯示名稱」）
- 右鍵對方鐵砧可取消置頂
- 隱藏對方鐵砧後，hover 顯示虛線指示器，可拖動定位
- 每位玩家獨立控制螢幕上各鐵砧的顯示/隱藏，不互相影響
- 伺服器關閉時立即清除螢幕上所有其他鐵砧
- 點擊「斷線」不再誤觸自動重連
- 名字、虛線、聊天輸入框離開 hover 後統一 400ms 才消失

### ⚙️ 設定調整
- 「我是美術」改名為「美術相關功能」，預設開啟

### 🔧 修復
- 修復公司 Proxy / SSL 環境下自動更新無法連線；下載失敗時提供瀏覽器手動下載
- 修復伺服器無法啟動的 import 錯誤
